# Numenta Anomaly Benchmark (NAB)

**What it is:** The Numenta Anomaly Benchmark (NAB) is a comprehensive benchmark for evaluating algorithms for anomaly detection in streaming, real-time applications. The dataset includes univariate time series from real-world sources like AWS CloudWatch server metrics, online ad click rates, NYC taxi demand, and machine temperature sensors. Each series has expert-labeled anomaly windows.

**Source:** [https://github.com/numenta/NAB](https://github.com/numenta/NAB)
